# two-integer-sum 
## Input 
Given an array of integers nums and an integer target
## Problem 
Return the indices i and j such that nums[i] + nums[j] == target and i != j.
## Output 
array of size 2
## Constraints 
2 <= nums.length <= 1000
-10,000,000 <= nums[i] <= 10,000,000
-10,000,000 <= target <= 10,000,000
