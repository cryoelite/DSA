# is-anagram 
## Input 
Given two strings s and t
## Problem 
return true if the two strings are anagrams of each other, otherwise return false.
## Output 
bool
## Constraints 
s and t consist of lowercase English letters.
## Observations 
## Solution 
TC: O(max(n,m)) SC: O(1) Time taken to solve: 15 mins
