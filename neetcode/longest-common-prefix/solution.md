# longest-common-prefix 
## Input 
You are given an array of strings `strs`
## Problem 
Return the longest common prefix of all the strings
longest common prefix: The longest prefix common to all strings.
For ex.:
Input: strs = ["bat","bag","bank","band"]

Output: "ba"
Input: strs = ["neet","feet"]

Output: ""
## Output 
A string with the longest common prefix, or "" if none
## Constraints 
1 <= strs.length <= 200
0 <= strs[i].length <= 200
strs[i] is made up of lowercase English letters if it is non-empty.
